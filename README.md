# CRUD de Usuários com NestJs e SQLite

## Descrição

Api CRUD feito com NestJS, SQLite e Prisma.

Requitos: NodeJS instalado

## Instalação

```bash
$ git clone https://github.com/mjrdev/crud-nestjs.git
$ npm install
```

## Rodar Aplicação

```bash
# desenvolvimento
$ npm run start:dev

# produção
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
